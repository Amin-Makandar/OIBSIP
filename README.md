# Unit Converter Application

## 📱 Overview

The **Unit Converter Application** is an Android application developed using **Java** and **XML** in **Android Studio**. It allows users to convert values between different measurement units through a simple and user-friendly interface.

This project was developed as **Task 1** for the **Oasis Infobyte Android App Development Internship (OIBSIP)**.

---

## 🚀 Features

* Convert Length units
* Convert Weight units
* Convert Temperature units
* Dynamic category selection using Spinner
* Automatic unit list update based on selected category
* User-friendly interface
* Input validation using Toast messages
* Instant conversion result display
